# 📊 InsightIQ

### AI-Powered Business Intelligence Platform

Transform business data into executive insights using AI, SQL, interactive dashboards, and natural language queries.

---

## 🚀 Overview

InsightIQ is an AI-powered Business Intelligence platform that enables business users to analyze sales performance without writing SQL. The application combines interactive dashboards, AI-powered business insights, natural language to SQL conversion, and executive report generation into one unified platform.

Users can filter business metrics, visualize KPIs, ask business questions in plain English, and receive strategic recommendations generated by an LLM.

---

## ✨ Features

- 📊 Interactive KPI Dashboard
- 🌍 Dynamic Region & Category Filters
- 📈 Interactive Plotly Visualizations
- 🤖 AI Business Consultant
- 💬 Natural Language → SQL
- 🧠 AI-Generated Executive Insights
- 📥 Excel Executive Report Export
- 📋 AI Explanation of SQL Query Results

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/arianna-vohra/InsightIQ.git
cd InsightIQ
```

## Create a Virtual Environment

```bash
python -m venv venv
```

### Windows

```bash
venv\Scripts\activate
```

### macOS / Linux

```bash
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Configure Environment Variables

Create a `.env` file in the project root.

```env
OPENROUTER_API_KEY=your_api_key_here
```

## Run the Application

```bash
streamlit run app.py
```

## 🔄 Project Workflow

1. User selects **Region** and **Category** filters.
2. Dashboard updates KPIs and visualizations dynamically.
3. User asks a question in natural language.
4. InsightIQ detects whether the query requires:
   - Dashboard analysis, or
   - SQL generation.
5. For SQL questions:
   - AI generates a SQLite query.
   - Query is executed on the database.
   - Results are summarized into executive insights.
6. Users can export a multi-sheet Executive Report in Excel format.
